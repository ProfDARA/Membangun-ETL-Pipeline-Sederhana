ETL pipeline ini terdiri dari 3 tahapan modul: extract, transform, dan load.
Setiap tahapan disimpan dalam file Python terpisah untuk memudahkan modularitas dan unit testing.
Main file akan mengorkestrasi keseluruhan proses ETL.




# Menjalankan skrip

pip install pipreqs

python main.py

# Menjalankan unit test pada folder tests

python -m pytest tests
